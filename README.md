# TP15

||Tung Tung Tung Math||
-----------------------------
Objetivo didáctico:
El objetivo del juego es que los niños practiquen operaciones básicas de matemática —(suma, multiplicación y división)— a través de desafíos breves y divertidos. Favorece el cálculo mental, la precisión y el razonamiento rápido.

||Contenidos escolares que aborda||
-----------------------------

•Cálculo mental con números naturales
* Sumas
* Multiplicaciones
* Divisiones exactas
•Estrategias para verificar resultados
* Resolución de problemas cortos
* Manejo de intentos y autocontrol del error

||Instrucciones de juego||
-----------------------------
1. Entrás al lobby del juego Tung Tung Tung Math.
2. Vas a ver tres puertas, cada una representa una operación matemática:

    ➕ Suma
   ✖ Multiplicación
   ➗ División
3. Elegí una puerta para comenzar.
4. Cada sala tiene 3 ejercicios para resolver.
5. Escribí tu respuesta y tocá Enviar.
6. Tenés 3 intentos totales para equivocarte.
7. Si completás las tres salas, aparece la pantalla de ¡Victoria!.
8. Si gastás todos los intentos, aparece la pantalla de Derrota, pero podés volver a jugar.
